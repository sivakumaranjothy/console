Visual Studio Code is a Code Editor that is a good use when it comes to debugging codes. Using it is very simple and similar to other Code Editors. There are seven step to set up Visual Studio Codes. First, Download and Install. Then,Create a new file. Next , See an overview of the user and install support for your favourite programming language. For the fourth step , change your keyboard shortcuts and easily migrate from other editors using keymap extensions. Then, customize your editor with themes. Finally , Explore VS Code features in the Interactive Editor Playground.

The console is the text output device for system administration messages. These messages come from the kernel, from the init system and from the system logger.It is used to maintain or monitor the status of the network or computer. To open the terminal in Visual Studio, choose View > Terminal.
